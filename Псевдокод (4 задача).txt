Реализация put() на псевдокоде:
Тип данных "целое число" хэш = хэш(ключ)
Тип данных "целое число" индекс = (n - 1) & хэш, где n = длина массива хеш-таблицы (обычно степень двойки)

Если хэш-таблица[индекс] == null: 
хэш-таблица[индекс] = новыйУзел(хэш, ключ, значение, null)

Иначе: 
 ключи совпадают (equals):
	староеЗначение перезаписывается новым
Иначе: 
 новыйУзел добавляется в односвязный список или красно-чёрное дерево (если цепочка слишком длинная)
// Коллизии происходят, если у разных ключей: 
// - одинаковый хэш, 
// - или разный хэш, но одинаковый индекс (из-за побитовой операции &))



Реализация get() на псевдокоде:
Тип данных "целое число" хэш = хэш(ключ)
Тип данных "целое число" индекс = (n - 1) & хэш

Если хэш-таблица[индекс] == null:
	вернуть null
Иначе: 
 перебор:
 	если хэш совпадает и ключи равны (equals()):
		вернуть значение

 вернуть null (если ничего не найдено)
